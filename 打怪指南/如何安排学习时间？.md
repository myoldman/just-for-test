# front

想要学有所成坚持到最后，你需要：

+ 每天投入 1+ 小时
+ 每周投入 10+ 小时
+ 整体 100+ 小时的全情投入 。全情是指至少 42% 的 `心流` 状态呢;-)





# back

更理想的时间投入是能空出固定的时段投入课程： 

+ 早上 0.5 小时
+ 晚上 1 小时
+ 周末 4 小时
+ 每周 10+ 小时

规律的作息是你面对未知世界强大的后盾！摸索出适合自己的输出节奏才是最重要的。

晚上异动较多，更建议尽量早睡早起，把输出时间安排在早晨哦，比如有的往期学员选择早起 2 小时写卡片。碎片时间也可以充分利用，比如坐地铁的时候刷刷卡片，写写小作业等。
